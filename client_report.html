General Experience and Reflections

What was the most challenging part of this project for you? Were you able to overcome it?

The most challenging part was learning how to use GitHub effectively while working with multiple team members. I also had some difficulty publishing websites with GitHub Pages. Through practice, troubleshooting, and asking questions, I was able to overcome these challenges and became much more comfortable using GitHub.

What did you enjoy most about this project?

I enjoyed working with other students and seeing how everyone's individual contributions came together to create a complete website. It was rewarding to apply the HTML and CSS skills I learned throughout the semester to a real team project.

What would you do differently if you could do the project again?

If I could complete the project again, I would communicate more often with my teammates, organize tasks earlier, and spend more time testing the website before publishing it. I would also use GitHub branches more effectively to keep the development process organized.

Do you feel more confident in working on real-world projects after this experience? Explain.

Yes. This project gave me experience working as both a client and a developer while collaborating with a team. I learned how to communicate requirements, use GitHub for version control, publish websites, and solve technical problems. These experiences have made me more confident in working on future web development projects.
